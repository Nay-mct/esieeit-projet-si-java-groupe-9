# Presentation du projet

Deux supports sont disponibles :

- `docs/presentation/presentation.html`
- `docs/presentation/PROJECT_OVERVIEW_SLIDES.md`

## Support recommande

Pour presenter, utilise surtout :

- `docs/presentation/presentation.html`

Cette version contient :

- un design de slides
- de la navigation clavier
- une barre de progression
- des schemas visuels sur l'architecture, le flux JWT et la persistance

## Comment l'ouvrir

1. ouvre `docs/presentation/presentation.html` dans un navigateur
2. mets le navigateur en plein ecran
3. navigue avec `fleche droite`, `fleche gauche`, `espace`, `Home`, `End`

## Version Markdown

Le fichier `PROJECT_OVERVIEW_SLIDES.md` reste utile si tu veux retravailler rapidement le contenu ou reutiliser l'extension Decker.

## Important

Sur cette machine, `pandoc` et la CLI `decker` ne sont pas installes.

Donc :

- le support visuel HTML est pret et utilisable
- l'export automatique en `.pptx` n'a pas ete genere ici
